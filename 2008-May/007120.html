Comment (by stippi):

 A mime database entry may specify several icons. Especially application
 entries may specify the icon of the application, plus one or more icons
 for associated files. Your code would only retrieve the application icon,
 since it ignores the "const char* fileType" parameter. As far as I am
 aware, every code that deals with icons, uses BIconUtils sooner or later.
