 Consider a axample:


 {{{
 #include <unistd.h>
 #include <search.h>
 int main() { return 0; }
 }}}

 and try to compile:


 {{{
 In file included from /boot/home/1.c:2:
 /boot/develop/headers/posix/search.h:35: conflicting types for `restrict'
 /boot/develop/headers/posix/search.h:35: previous declaration of
 `restrict'
 }}}

 Changing restrict to __restrict fixes the problem, but I'm not sure that
 is a good solution.
