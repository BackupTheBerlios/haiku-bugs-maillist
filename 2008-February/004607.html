Comment:

 I believe this problem is a result of Haiku not yet supporting transparent
 syscall restarts. Resizing the terminal window will cause a SIGWINCH to be
 sent to ftp, which interrupts a send() or recv().

 Assigning this to myself for the time being, so that it is on my task list
 at least. If someone else wants to tackle it, feel free...
