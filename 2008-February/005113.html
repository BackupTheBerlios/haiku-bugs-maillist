 Building and installing Haiku on a partition from Linux using the cross-
 tools gcc2.

 The machine is an AMD Athlon64 3400+ with an nVidia 7950GT PCIe card and a
 Dell 2407 widescreen display.

 Upon initial boot up the Haiku logo displays, but afterward the blue
 background and cursor do not appear, instead the screen goes black. The
 hard drive light indicates that the system is continuing to load.

 However, if the system is booted using an integrated nVidia 6150 chipset,
 it exhibits no problem at all.

 Subsequently, if the system is booted once again with the 7950GT card
 AFTER it has successfully booting using the integrated 6150 chipset, the
 display will function normally.
