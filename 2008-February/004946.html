 to the image. (sigsuspend.diff, HaikuImage-sigsuspend.diff

 exist the line:
 HAIKU_ADD_POSIX_TEST_SUITE_TO_IMAGE = 1 ;
 * build the image, boot haiku and
 go to the /boot/home/posixtestsuite/ directory
 * run ./run_posix_tests all
 * you should go to the kdl screen, when the sigsuspend_6-1
 is executed with a similar bt to
 sigsuspend-kdl-bt.txt
