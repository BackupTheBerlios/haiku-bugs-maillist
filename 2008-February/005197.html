Comment (by mmlr):

 Then it is most probably related to USB legacy emulation. In case you
 remove all USB files, the stack and host controller driver obviously will
 not get loaded. In this case the controllers will stay in legacy emulation
 mode. Maybe the PS/2 bus manager has a problem in that case? You could try
 removing the PS/2 bus manager too and see if this makes any difference.
