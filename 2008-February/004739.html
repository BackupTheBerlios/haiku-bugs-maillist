 The NIC on my machine seems to be supported as well as I can tell after
 this simple patch that enables the additional device id to the rtl8169
 driver.

 I'm not sure if the device id my card uses was specifically excluded
 because of some incompatibilites or not, but I would assume the card NIC
 wasn't available at the time the author wrote the driver.

 The currently supported device ids of the driver are 0x8167 & 0x8169, this
 patch allows 0x8168 as well.
