 Revision: r23905. Reproducible on real hardware and in VMware.

 How to reproduce:
  - boot Haiku
  - start LaunchBox via Deskbar -> Desktop Applets -> LaunchBox
  - click Terminal to activate it
  - switch to workspace 2 via Alt-F2
  - LaunchPad window is not shown, although per default it is "Show On All
 Workspaces".

 The behavior seems to depend heavily on what window is activated, maybe
 even the order in which they've been activated/deactivated.
