#1326: Undo/Redo menu items are not properly enabled/disabled in Mail app
---------------------------------+------------------------------------------
 Reporter:  ksmith               |       Owner:  axeld         
     Type:  bug                  |      Status:  new           
 Priority:  normal               |   Milestone:  R1            
Component:  - Applications/Mail  |     Version:  R1 development
 Platform:  All                  |  
---------------------------------+------------------------------------------
 When you launch the mail app, the redo menu item in the Edit menu is
 enabled. This should not be enabled. There is nothing to redo. The undo
 menu item is properly disabled at this point. Type a character in the
 message body. Undo is now enabled. Select undo, which will remove the
 character you typed. The undo option remains enabled, even though there is
 nothing else that can be undone. The undo option should become disabled
 when there is nothing remaining to be undone.
