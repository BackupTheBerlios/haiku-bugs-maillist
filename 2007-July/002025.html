 Creating an e-mail account in the E-mail preferences does not create the
 folder named mail in /boot/home. Without this directory, the mail app will
 fail to save drafts. I am unsure if this affects the receipt of emails.

 The mail folder should be created upon the creation of an e-mail account
 or should just simply exist.
 The mail app should have the ability to create the folder if it should not
 exist for some reason.
