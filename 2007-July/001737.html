Comment (by bonefish):

 AFAIK there's no boot loader for Haiku in the works, ATM. The BeOS bootman
 won't work under Haiku, since it seems to use a private API, that we won't
 provide.

 Apart from that, every other boot manager -- including BeOS' bootman --
 should be able to boot Haiku (via chain loading), if the partition has
 been made bootable correctly (via makebootable -- in doubt use Haiku's
 version).
