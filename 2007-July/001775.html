Comment (by mmlr):

 You can check if they are hubs using the device entries exported by
 usb_raw. Run "ls -laR /dev/bus/usb" with and without the replicator and
 compare the listing. You should see something like "/dev/bus/usb/0/0/hub"
 when it is a hub that is connected. You can also verify if the devices
 plugged into a hub work by using this method. The entries are then
 numbered according to the port they are plugged into.
