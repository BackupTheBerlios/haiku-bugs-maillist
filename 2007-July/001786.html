 There is no way to unnattach an image that has been attached via
 img_attach, see [http://dev.haiku-os.org/ticket/1294 ticket 1294] for more
 issues with image_attach. [http://drupal.org/node/98154 This post] on
 drupal.org also documents this bug, which is solved in version 5.0 or
 avoided by using img_assist instead of image_attach. The solution posted
 on drupal.org has no effect for our installation - my "fix" was to upload
 a transparent png (hack.png) and allow re-attachment of images.
