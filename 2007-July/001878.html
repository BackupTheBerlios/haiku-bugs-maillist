 The spell check feature does not work because the words file does not
 exist. The mail app expects the words file to be located at
 /boot/optional/goodies/words. The words file is in the haiku source at
 /haiku/src/apps/mail/words but it does not get built into haiku. Placing
 the words file in the correct directory in haiku makes the spell check
 feature work like a charm.
