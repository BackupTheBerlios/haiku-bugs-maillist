 Attempting to play any video with "Hardware Video Overlay" enabled will
 not only not work (black screen with audio), but cause App Server to
 become glitchy enough that a hard reset is usually required.
 Specifically, it uses 100% CPU and is slow to respond to any user
 interaction, though it returns to ~0% CPU usage when idle.

 This is on a GM45 chipset with an Intel 4500MHD graphics card.
