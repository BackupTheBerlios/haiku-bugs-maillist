 During boot screen, icons light up properly [Picture1]. After last icon is
 lit, entire boot screen shifts to the left (Haiku logo is at left edge of
 screen) [Picture2].  The mouse appears, the hard drive indicator continues
 for a few seconds as booting completes.  The mouse can be moved but is
 very slow / choppy [Picture3].  Some artifacts from the underlying desktop
 can be revealed by moving the mouse.  The system seems to be responding to
 keyboard input but without visual reference it is tough to tell what is
 happening.  Booting this system using safe mode graphics at 1024x768x32
 brings up a usable system but with vesa mode graphics.  This laptop also
 boots up under BeOS 5.0.3 with Radeon driver with no issues (even dualhead
 support). There is no way to specify the correct startup resolution for
 the Radeon driver in Haiku.  I have included an output from listdev and
 the referenced pictures in the above text.
