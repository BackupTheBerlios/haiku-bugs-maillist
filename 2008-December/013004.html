Comment(by stippi):

 It's an app_server crash, not a KDL. And the back trace is unfortunately
 not useful, since it's for the wrong thread (the main thread). To be on
 the safe side, if you see such a trace again, try entering the command
 "cont" into the GDB terminal. That should then drop into the thread that
 crashed and you can type bt again.
