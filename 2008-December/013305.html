Comment(by sil2100):

 Ah, I understand very well what you are talking about. This was one of the
 few things that were heavily discussed during the design stage of the
 PackageInstaller. I originally wanted to have such a feature, since I saw
 it very very useful, but since the installer was planned to be as simple
 as possible, the idea was eventually dropped. If I remember correctly
 though, we agreed on creating another application someday, called
 PackageInspector, which would have such functionalitty (as well as many
 more). The idea was to have a different application for browsing through
 all available package information, just like files to be installed,
 version, description and so on. But eventually, this is yet waiting for
 implementation.

 But someday I will most certainly get back to this project - along with
 finishing the InstalledPackages application (for browsing and
 uninstallation).
