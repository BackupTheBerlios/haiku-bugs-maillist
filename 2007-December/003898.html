Comment (by mmlr):

 Replying to [comment:5 kaoutsis]:
 > i am confused, how many cpus does this machine have,
 > one or two? :-)

 It has one physical processor but when HT is enabled it provides two
 logical processors.

 When you enable hyper-threading there are two processors listed in the
 ACPI table. That it works with hyper-threading enabled is caused by the
 fact that with the revision you test there is a limit of 2 CPUs. With
 enabled HT the two logical processors are enumerated first and the
 "strange ones" that cause the hang are disabled because of the limit. I
 have removed that limit some revisions later though so if you update to a
 more recent one and try again you will see it fail with either HT setting.

 There is an enable flag in the local APIC structure which I forgot to
 check against. If the entries your BIOS reports are disabled, it is an
 easy fix. If not I have to investigate this further. Will check in a fix
 later tonight.
