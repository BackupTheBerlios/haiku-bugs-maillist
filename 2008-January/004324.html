 The GL dispatcher should route the call to :
 src/kits/opengl/mesa/swrast/s_readpix.c function _swrast_ReadPixels()
 The Mesa function should then call the render buffer function
 fRenderBuffer->GetRow().
 This one can be get_row_RGBA32(), get_row_RGB32() or get_row_RGB16().
