 This is in vmware, r22714.

 When you do a query for any filename containing "über", the generated
 formula is (name=="*ü[bB][eE][rR]*"). The "Umlaut" is not case-
 insensitive. It should be
 (name=="*[üÜ][bB][eE][rR]*")
