 It would be nice if Haiku supported the latest broadcom gigabit ethernet
 chips. Currently Nathan's [http://bebits.com/app/3909 driver] is in SVN,
 but it does not support new chips like the one in my laptop (BCM 5787M)
 Unfortunately the code his driver is based on was not updated much, and
 linux switched to a new codebase. So I see 2 options:

 Port the
 [http://git.kernel.org/?p=linux/kernel/git/stable/linux-2.6.23.y.git;a=blob;f=drivers/net/tg3.c;h=9034a05734efb86b503bfa01f8d6673f071ce1de;hb=HEAD
 linux driver] or use the FreeBSD network driver compatibility layer to use
 the [http://www.freebsd.org/cgi/cvsweb.cgi/src/sys/dev/bge/ FreeBSD
 driver]. Most likely the last option would be easiest.
