Comment (by jackburton):

 Looks like TermView forwards those key to bash, so it's bash which doesnt'
 handle them.
 A very quick search on the internet resulted in this:

 in /etc/inputrc:
 # allow the use of the Home/End keys
 # "\e[1~": beginning-of-line
 # "\e[4~": end-of-line Uncomment the two lines and all is well.
