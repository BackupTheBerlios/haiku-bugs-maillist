#1146: The auich driver miscounts on controls, turns of amp early, never turns it
of on shutdown, and UI improvement
--------------------------------+-------------------------------------------
 Reporter:  tqh                 |       Owner:  korli         
     Type:  bug                 |      Status:  new           
 Priority:  normal              |   Milestone:  R1            
Component:  Drivers/Audio/AC97  |     Version:  R1 development
 Platform:  All                 |  
--------------------------------+-------------------------------------------
 The auich driver makes noise on shutdown because the amplifier is left on
 after shutdown, the amplifier is also turned on directly after init. The
 patch changes this to be the last thing to be turned on, and the first to
 be turned of.

 The creation of controls had an of by 1 error where the last control was
 skipped.

 Also made the controls more UI friendly so that the 'boost mic +20dB' is a
 part of the mic control instead of it's own tab, and the AC97 mixer tab is
 changed to be the first one as it is probably more used than the 'Record'
 tab, which now is called 'Recording'.

 Very nice driver though...
