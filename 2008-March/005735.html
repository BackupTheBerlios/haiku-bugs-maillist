 Did you ever run low on memory during working? Were these newly created
 files?
 Because if you didn't, the files would be kept in memory all the time, and
 therefore, something would actively clobber it.

 If they were new files and memory was low, eventually something marked the
 pages as clean, so they would never be written back to disk. Does that
 garbage look like something that might have been on disk previously?
