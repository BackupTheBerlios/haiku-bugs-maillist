 The new disk system seems to have broken reading CDs in some respect. If I
 insert a zeta CD either during boot or while running, Haiku sees the boot
 floppy image track, but instead of seeing the BFS track shows another copy
 of the Haiku hard disk volume in the partition list, with no way to access
 the actual BFS track on the CD. This behavior is consistent. Prior to the
 disk system, it was able to see both tracks properly, but only if the CD
 was already in the drive while booting.
