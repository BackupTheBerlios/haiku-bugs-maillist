Comment(by mmlr):

 If you're building a new image to dd to the medium then you could remove
 "<usb>ehci" from build/jam/HaikuImage. If you're updating an existing
 image/filesystem then you need to remove system/add-
 ons/kernel/busses/usb/ehci. You could do that using the mount pseudo-
 target like "jam @yourTargetBuildProfile mount" which should then bring up
 the target in the fsshell. From there you can remove said file.
