 This is r33755.

 If you right-click a file while holding down SHIFT, the context menu shows
 slightly different commands. Among them "Force Identify" for correcting
 wrongly set filetypes.

 "Force Identify" only works when you hold down SHIFT while invoking the
 menu item.

 It should've been enough to hold SHIFT while right-clicking for the
 context menu to make the command "Force Identify" available. Selecting
 "Force Identify" from it should then work without holding SHIFT.
