 To work with posix threads struct sched_param must be defined.
 /boot/develop/headers/posix/sched.h is missing this definition.

 adding :
 struct sched_param { int sched_priority; };

 is all that is needed.  It seems that the spec only defines one field.
 Any others are platform specific.
