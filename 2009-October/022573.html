Comment(by devine):

 The reason this was not done initially was because there is accessibility
 issues with 7zip. A normal .zip file presents a low barrier of entry for
 users of all experience on all platforms. Using bz2 would be fine for
 users on Unix-like OSes but I doubt that many Windows folk have ever seen
 that file type.

 On the other hand, the compression is really awesome!

 Perhaps the barrier can be lowered by making it very clear in multiple
 places that 7zip is open source (free) and can be downloaded via a link. A
 test could be performed with fairly computer illiterate users to determine
 the viability.
