 This is r33529.

 If you've set the Mail preference "Automatically mark mail as read" to
 OFF, you'll get another down-arrow symbol in the toolbar when opening a
 mail: "Read".[[BR]]
 It's supposed to close the current mail and set its status to "Read" and
 move to the next new mail. Only, it doesn't move to the next mail, the now
 "Read" mail window stays open.
