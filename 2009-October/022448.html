Comment(by kwurzel):

 We could just use the (styled) output of Apache's mod_autoindex (see
 http://recurser.com/articles/2008/12/29/styling-apache-directory-listings-
 with-mod_autoindex/ for an explanation). I think Canonical uses it at
 http://releases.ubuntu.com/9.04/ for example.

 Alternatively, we could use a simple PHP Directory listing script like
 http://www.evoluted.net/community/code/directorylisting.php

 I would prefer the first one, as it is much faster (at least in theory)...
