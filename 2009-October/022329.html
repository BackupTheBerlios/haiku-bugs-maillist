 How to reproduce:

 1) Open several Tracker windows.
 2) Open several applications which have only one window (ex. Pulse + Clock
 + Playground).
 3) Summon Twitcher (press and hold Ctrl+Tab or Alt+Tab if you use
 Windows/Linux mode for shortcuts). Don't release Ctrl (Alt) yet.
 4) Press Tab to switch to application with several windows (Tracker).

 Note there is an arrow pointing down which indicates an app (Tracker) has
 more than one window you can chose from to switch to.

 5) Press Tab again several times.

 Note that arrow is still present for applications which have only one
 window. It's not filled with dark color and will became less and less
 visible as you switch from one app to another.
