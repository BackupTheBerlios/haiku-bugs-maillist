 Don't know if it's a bug or missing feature. Tracker updates info only for
 files created before the folder was opened.

 How to reproduce:

 1) Open existing or create new folder.
 2) Set folder view to List View.
 3) In this folder create some file (ex. StyledEdit text file).
 4) Change this file (add some text and save changes).

 Note that in Tracker window there are no changes, file size still 0 bytes
 and Modified time still old.

 Close folder and open it again. Now file size and modified time are
 updated and will update realtime after any change.
