 Reporter:  v        |       Owner:  haiku-web     
     Type:  bug      |      Status:  new           
 Priority:  normal   |   Milestone:  R1

Blockedby:           |    Platform:  All           
 Blocking:           |  
---------------------+------------------------------------------------------
 http://www.haiku-
 os.org/documents/dev/driver_development_presentation_waltercon_2006
 [[BR]]
 Instead of the non-alphanumeric characters the HTML escape sequences for
 them are _shown_.[[BR]]
 [[BR]]
 I'd guess this isn't the only page on which this occurs, so maybe this can
 be solved without awaiting a report for each page.[[BR]]
 [[BR]]
 I'll go into specifics now:[[BR]]
 [[BR]]
 under heading "1. Device File System"[[BR]]
 under heading "For development:"[[BR]]
 [[BR]]
 in the sentence[[BR]]
 ''use rescan <driver name> to make sure the system reloads your
 driver after a recompilation''[[BR]]
 [[BR]]
 should be '''<driver name>''' instead of '''<driver name>''' [[BR]]
 [[BR]]
 under heading "2. Modules"[[BR]]
 [[BR]]
 in the first code block
 {{{
         module_info *modules[] = {
                 (module_info *)&scsi_for_sim_module,
                 (module_info *)&scsi_bus_module,
                 (module_info *)&scsi_device_module,
                 (module_info *)&scsi_bus_raw_module,
                 NULL
         };
 }}}
 should be
 {{{
         module_info *modules[] = {
                 (module_info *)&scsi_for_sim_module,
                 (module_info *)&scsi_bus_module,
                 (module_info *)&scsi_device_module,
                 (module_info *)&scsi_bus_raw_module,
                 NULL
         };
 }}}
 in the second code block[[BR]]
 [[BR]]
 the lines
 {{{
                 (module_info **)&gManager);
         if (status < B_OK) {
 }}}
 should be
 {{{
                 (module_info **)&gManager);
         if (status < B_OK) {
 }}}

 in the third code block[[BR]]
 the lines
 {{{
                 {B_DEVICE_MANAGER_MODULE_NAME, (module_info
 **)&gManager},
                 {LOCKED_POOL_MODULE_NAME, (module_info
 **)&gLockedPool},
 }}}

 should be
 {{{
                 {B_DEVICE_MANAGER_MODULE_NAME, (module_info **)&gManager},
                 {LOCKED_POOL_MODULE_NAME, (module_info **)&gLockedPool},
 }}}

 under heading "4. Interrupts"[[BR]]
 [[BR]]
 the first code block
 {{{
 status = install_io_interrupt_handler(info.pci->u.h0.interrupt_line,
                 &my_interrupt_handler, (void *)&info, 0);
 }}}

 should probably be

 {{{
 status = install_io_interrupt_handler(info.pci->u.h0.interrupt_line,
                 &my_interrupt_handler, (void *)&info, 0);}}}
