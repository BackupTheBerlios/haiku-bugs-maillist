 I found that my (original) Intuos 6x8 USB didn't function properly under
 Haiku (though it has always worked fine with Stippi et al's code in BeOS).
 Looking at the source I saw that the pen-contact (Button 1) detection had
 changed from the earlier code, and no longer matched the Intuos protocol.

 While fixing this, I did some reverse engineering, and found out how the
 Eraser function is signalled.  (This was never implemented in the code.)

 So I made a patch (attached) that gives my tablet full function.  I think
 it should be compatible with the other models handled by the same code,
 but of course this needs to be checked.

 There are a couple of fiddle-factors that are right for my tablet (contact
 detection and proximity), but I don't know if there's likely to be any
 problem with them for others.
