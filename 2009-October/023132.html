Comment:

 Same problem for me. Two hard drives on ICH8-M AHCI, one optical drive on
 ICH8-M ATA. Also tried creating partitions with parted, but no success.

 Workaround: temporarily setting the controller mode from AHCI to IDE (if
 you can); then BootMan is able to properly install. After that, reboot and
 set the mode back to AHCI. BootMan will work and boot the system just
 fine. This worked for me.
 So maybe this is just an issue with the BootMan installer and not with the
 actual bootloader itself?
