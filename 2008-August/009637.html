 {{{
 for i in `seq -w 1 1000` ; do mkdir $i ; cd $i ; done
 }}}

 deepth of 204 levels, the script continues to create directory in the last
 level, but it is unable to cd'in, and returns
 an error of "no such file or directory".

 While not critical, it shouldn't create invalid directory.
