 Idea partly by Koki in #haiku

 Add preflet descriptions to a Description attribute and display them in
 the Tracker window.

 Set the default view for the preferences folder to detailed list view,
 hide every column except Name and Description (of course, the user can
 change this), as size and date &c. are not as important for preflets. This
 provides quick, easy access to the preflets and what they all do without
 having the user do any extra work (just open the Preferences folder)
