Comment(by axeld):

 Tracker memorizes the window layout exactly as you specified it. This has
 the advantage that it will also work with removable media, even if you
 don't have the MIME type defined, or changed its attributes.

 Of course, the downside is that changes aren't immediately visible. But in
 reality, I don't think one changes attributes that often, so that the
 persistence feature wins.
 And when it's properly documented, it should also be much less irritating.
