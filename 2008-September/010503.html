Comment(by axeld):

 The video mode selection is only for booting and VESA (well, not really
 anymore, since Jan's VBE mode switch changes). Only if you select "fail
 safe video mode", the graphics driver is not used, but VESA instead.

 One very important info is missing from this report, though: before that
 revision, it used 1600x1200, not 1920x1280. That means this revision only
 made it no longer ignore the native monitor resolution, but the Intel
 driver apparently has problems to switch to that resolution with the i865.

 Unfortunately, I don't have a monitor that could display this resolution,
 so I'm not sure how I can look into it.
