Comment:

 You have to enable patented code in libfreetype for it to work with
 hinting on. See headers/libs/freetype2/freetype/config/ftoption.h for more
 info. In another words, you have to build your own Haiku image, since this
 option is not enabled in the source repository. The preflet should mention
 this, so I leave the bug open.
