Comment(by umccullough):

 Replying to [comment:5 monni]:
 > For it to work, I need to run it as user "root"... Normal user won't
 work as the whole /dev file system (udev) is created run-time... There is
 no way to permanently chmod or chown individual "nodes".

 Which is why many of us just chmod the necessary paths before jamming an
 image :)

 BTW, after you've been running your haiku builds as root - trying to run
 them as a normal user often fails to remove files created in the generated
 folder as root - so you'll want to chown those, or just remove them all
 and start clean.
