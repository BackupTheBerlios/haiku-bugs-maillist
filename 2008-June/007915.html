 While testing with r25575,hot-plugging a USB digital camera (Kyocera
 Finecam SR3) causes the system to freeze. The camera has a SD card fat
 type file system.
 To reproduce:
 1) Plug in the USB lead from the canmera.
 2) Try to move the mouse pointer and it is frozen
 3) Try to use the keyboard eg (CTRL ALT Delete) or any key and it is
 frozen.
 On the mount settings I tried selecting "Don't Automount". That  made no
 difference.
 With this same camera it can be mounted in BeOSR5.05 Bone without a system
 freeze, but I have to use DriveSetup to mount it from, (cannot use desktop
 right-click menu.
