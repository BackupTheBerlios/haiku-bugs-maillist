Comment(by umccullough):

 Not that I necessarily believe that it would work, but did you add your
 device to both the kernel driver and the accelerant?

 You must add it here:

 http://dev.haiku-os.org/browser/haiku/trunk/src/add-
 ons/kernel/drivers/graphics/nvidia/driver.c

 and also here:

 http://dev.haiku-os.org/browser/haiku/trunk/src/add-
 ons/accelerants/nvidia/engine/nv_general.c

 The second one is important because it tells the accelerant what features
 the chip supports and how to work with it.

 Best solution is to find one already there with the same features/revision
 as yours and add it as an additional case there. (perhaps along with the
 0x039110de which is also labeled as a 7600GT)

 Anyhow, if that still doesn't work - i'd guess Rudolf would have to
 provide more advice :)
