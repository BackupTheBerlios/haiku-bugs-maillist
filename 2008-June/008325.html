Comment(by idefix):

 It's not a rounding problem.

 When you open the VirtualMemory application for the first time and only
 check the 'Enable Virtual Memory'-box (do not move the slider), the
 application will create (on close) a 'virtual_memory'-file with the
 following contents:
 {{{
 vm on
 swap_size 0
 }}}
 The next time you open VirtualMemory, it will interpret this as Virtual
 Memory disabled.
