Comment(by mmlr):

 Oh, you just installed that BeOS package? That could cause a few problems
 indeed, as it contains an updated pipefs replacing the one present. This
 was intended on BeOS because the pipefs there has issues. But on Haiku
 it's not really a good idea to replace it, as this most certainly isn't
 compatible with the current filesystem API. You should try a package that
 only supplies the subversion binaries. For example you could try
 http://haiku.mlotz.ch/subversion-1.1.3-bone.zip or the more current
 http://haiku.mlotz.ch/subversion-1.4.6-bone.zip. Note that these support
 svn, http and https, but not svn+ssh, as the apr_proc_stub and some code
 utilising it is missing from them.
