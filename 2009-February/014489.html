Comment:

 FAT is pretty restrictive when it comes to which characters it accepts in
 file names.
 This is a file system deficiency that we can't do anything about, though.
 I've built a temporary hack into Tracker that would automatically
 translate names when copying to FAT (it's a build time option IIRC).

 In the future, we should let the file system itself publish the characters
 it supports, and how large a name can be, or, alternatively, it should
 have a function to correct names. Then, Tracker could just use this
 function, and could ask the user, if the file names that need a rename
 should be renamed automatically.
