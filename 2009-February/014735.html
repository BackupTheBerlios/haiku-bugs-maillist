 Use Deskbar to debug some thread. Now perform a command which results in
 long output, "threads apply all bt" should do the trick. You need an
 output sufficiently long to get a
 {{{
 --- Type <return> to continue, or q <return> to quit---
 }}}

 Leave gdb in this state and use Terminal->Quit in the debug window.

 Experienced behaviour: the terminal closes, but the debugged team is now
 frozen and the gdb team is still running.
 Expected : terminal closes, gdb terminates, debugged team continues to
 run.
