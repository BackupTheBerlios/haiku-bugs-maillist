Comment(by mmlr):

 People, don't panic. The optional package hasn't been updated yet. The GCC
 build system, being quite complex, currently triggers a severe bug in
 Haiku that I'd like to find and fix. So my priority is to fix the problem
 and not to actually build a new GCC. It will be made available soon,
 probably during the next week. For now if you're really desperate you can
 still overwrite the specs by supplying a specs file and applying the
 changes from r29297 to that one:

 {{{
 gcc -dumpspecs > /boot/develop/gnupro/lib/gcc/i586-pc-haiku/4.3.3/specs
 open /boot/develop/gnupro/lib/gcc/i586-pc-haiku/4.3.3/specs
 }}}
