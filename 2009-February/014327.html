 If you forget that a partition is mounted as read-only and you try
 deleting multiple files with SHIFT+DEL, then Tracker will attempt to
 delete the files.

 Tracker first generates the pre-task calculations.
 Then for each file that is attempted to delete, Tracker will issue an
 error-stlyed alert box.  The only way to cancel this action is to
 forcefully kill Tracker.

 r29153, real hardware.
