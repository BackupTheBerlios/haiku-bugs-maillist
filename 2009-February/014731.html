 allow a user to :
 -select gcc version (GCC 2, GCC 4, GCC 2+4, GCC 4+2)
 -input kernel-version (aka uname -v)
 -hardware type (real, virtualized:[qemu, ...,])

 These additional fields should help organize bug reports.
