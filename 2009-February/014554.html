 When you move a disk icon from one location on the desktop to another
 location on the desktop, the mouse cursor changes to a copy cursor. When
 you release the mouse button, the disk icon isn't copied but moved to the
 new location. The mouse cursor should therefore not change to a copy
 cursor, but stay the same.

 See attached screenshot for the different mouse cursors when dragging
 different items to a new location on the desktop.
