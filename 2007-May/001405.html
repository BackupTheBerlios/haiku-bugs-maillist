 When adding a PCL6 printer, an additional dialog shows up to select the
 protocol.  The problem then occurs when the user presses the cancel button
 or the close button on the tab.  It asks whether to make the printer the
 default and doesn't add the printer when answering that dialog.

 When canceling this dialog, I would actually expect it to go back to the
 first add printer dialog.  It might be acceptable to just have it avoid
 bringing up the question about making it a default printer.
