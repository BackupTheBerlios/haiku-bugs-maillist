Comment (by shinta):

 Thank you for your investigation.

 I think that archived class name and instantiate class name are
 different.When I modified TextInput.cpp as follows,
 BTextControl::BTextcontrol(BMessage*) works well.

 _BTextInput_::Instantiate(BMessage* archive)
 {
 if (validate_instantiation(archive, "BPrivate::_BTextInput_"));
 ...(omit)...
 }

 Currently, archived BMessage has the class name "BPrivate::_BTextInput_".
 But, archived BMessage in ZETA (and maybe in BeOS, too) has the class name
 "_BTextInput_".

 I think it is better to archive with the name "_BTextInput_" for backwards
 compatibility, then we don't need to modify _BTextInput_::Instantiate().
