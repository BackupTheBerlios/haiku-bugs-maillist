Comment (by mmlr):

 From the serial log I take it you are running r24782. Those revisions have
 serious issues with the block cache (see bug #2008) which most probably
 cause these KDLs. You should update to a current revision that has those
 issues fixed first and then try that again. The KDL and serial log at
 least do not indicate an actual problem with usb_disk.
