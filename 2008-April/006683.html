 The exit code of {{{which}}} is not inline with the expectations of some
 {{{./autogen.sh}}} and similar scripts.

 {{{
 which someexistingtool || echo "You need someexitingtool >= 0.2"
 }}}
 is expected to print the message iff someexistingtool is not found. On
 Haiku it is sometimes inversed.
