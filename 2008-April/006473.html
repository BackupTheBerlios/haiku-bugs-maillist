Comment (by bonefish):

 Replying to [comment:3 axeld]:
 > There is no BVolume::Unmount(), actually. Tracker just deletes the mount
 point manually. Of course, if Mount() creates the mount point, Unmount()
 should remove it - otherwise it really shouldn't, though (as you might
 want to mount a volume not on root level).

 Mount() gets the mount point as optional argument. If one is given it
 expects it to exist. If not given, the standard mount point is created.
 Hence my suggestion to add a flag to Unmount().
