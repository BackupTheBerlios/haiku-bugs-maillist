Comment:

 This is not a race condition or anything, it's a plain simple timeout that
 happens. The device does not respond quickly enough therefore usb_disk
 will abort and return a timeout error ("error: Operation timed out"). The
 timeout I (arbitrarily) set is 2 seconds right now, which might just not
 be enough. Please increase the timeout in src/add-
 ons/kernel/drivers/disk/usb/usb_disk/usb_disk.cpp line 145 by a second
 until you don't see this happening anymore and report how much it needs.
 Other subsystems seem to use timeouts of up to 20 seconds, but I don't
 really want to increase it that much. Maybe 3 or 5 seconds would be
 acceptable though.
