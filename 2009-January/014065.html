 The nVdidia card has two DVI ports. When a TV is connected (it does not
 have to be on or in stand-by) to the second DVI port, the list of
 available screenmodes in the Screen Preferences tool is incomplete.

 When the TV is connected, the list of available modes is:
  * 640 x 480
  * 800 x 500
  * 800 x 600
  * 1024 x 640
  * 1024 x 768
  * 1280 x 720
  * 1280 x 768

 When the TV is not connected, the list of available modes is:
  * 640 x 480
  * 800 x 500
  * 800 x 600
  * 1024 x 640
  * 1024 x 768
  * 1152 x 864
  * 1280 x 720
  * 1280 x 768
  * 1280 x 800
  * 1280 x 1024
  * 1400 x 1050
  * 1440 x 900
  * 1680 x 1050 (native resolution of my monitor on DVI port one)

 The TV's native resolution is 1366 x 768, if that matters.

 Note that I am not (yet) running in dual-screen mode. The TV is connected
 to the nVidia card, but that's it.

 Haiku r29009 and NV43 [GeForce 6600 GT] (AGP)
