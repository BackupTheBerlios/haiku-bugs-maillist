 BRIDGED, but I can not connect to Internet (just ping to ADSL modem is
 OK). When I start Firefox 2.0.0.17 (Bon Echo), it can not display. Just
 the title and the outter frame of the Bon Echo window. I just want to read
 Be Book, but it is impossible.
 When I change the VMWare network setting to use NAT, I can use Bon Echo
 (Firefox) to browse the Internet.
 Now, I switched back to BRIDGED Network Adapter, and can not connect to
 Internet again, but still can read Be Book.
 So, I think there is something in Firefox that involved with the network
 setting, that thing MUST INITATE with NETWORK SETTING is OK.
 This was test with Haiku r28827, r28847 and Senryu Haiku r28815, with
 VMWare on Intel Core 2 Duo E2160.
