 This is sort of an odd situation, but when you resize either DeskCalc or
 Activity monitor to the full size of the screen the system memory usage
 goes up by 3MB. I assume it's due to the system creating redraw buffers
 for those windows.

 The issue is that when you return those windows to a respectable size
 their memory usage stays at what it was for when they were full-screened;
 the memory for those buffers doesn't seem to be freed again. Just for fun,
 try stretching the activity monitor a few times larger than the screen. TV
 doesn't have the same problem. In fact, it's memory usage didn't climb at
 all.
