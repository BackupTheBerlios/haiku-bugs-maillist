 These are proposed extensions to the Workspaces app to allow a more
 flexible behavior.

 Three command-line or context menu options

 --vertical
 --horizontal
 --grid X,Y

 Vertical and horizontal force a single line of workspaces in the specified
 orientation.
 Currently, the only way to get a horizontal line is to use an odd-number
 of workspaces and there is no way to allow a vertical arrangement.

 Grid X,Y will allow the user to specify a non-default grid arrangement.
 For example, with 16 workspaces, the default grid would display as 4x4.
 Grid 8,2 would display 2 rows of 8 workspaces.
