Comment(by bga):

 This is expected. Every BFS partition has a Desktop dir. And a Desktop dir
 is, by default, a merging off all Desktop dirs. So what you are seeing are
 the contents of this virtual Desktop dir which will be the contents of the
 Desktop dir in your boot partition (as the Desktop dir in the USB stick is
 empty).
