 Warning : Touchpad support should be enabled in the source tree to
 reproduce this problem.

 When rebooting from Haiku to BeOS, the mouse cursor is not moved
 correctly.

 The touchpad driver should send again standard ps2 packet when leaving.

 This is the purpose of the attached patch.

 Note : it will become usefull only after resolving the TODO in
 src/system/kernel/shutdown.c (close all services before leaving).

 Tested using this command line :

 kill <id_AppServer> <id_InputServer>; shutdown -r
