 It would be very nice and convenient if installer would contain two
 additional options:

 1) Clean Install

 This option should remove all Haiku system files present except user data
 (i.e. /boot/home) and super sugar would be an additional checkmark to also
 remove the user config directory to prevent malfunctions from stale config
 files.

 2) Archive and Clean Install

 Same as above, except that everything is archived first, either in a real
 archive or for speed reasons just moved to a restore folder whatever.
