Comment(by maxime.simon):

 DataTranslation now works with both layout and non-layout
 translators.[[BR]]
 For layout translators a piece of code is needed in the ConfigView:
 {{{
 BFont font;
 GetFont(&font);
 SetExplicitPreferredSize(BSize((font.Size() * 400)/12, (font.Size() *
 200)/12));
 }}}

 We ask for the an explicit preferred size which we use in DataTranslation
 to resize the window. This size has to be font sensitive (like in the
 piece of code above).
