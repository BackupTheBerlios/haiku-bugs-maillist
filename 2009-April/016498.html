Comment(by bga):

 Yes, I can confirm this is happening and I was able to partially track it
 down: What happens is that when yo are in a 64 bit host, the --use-32bit
 is being completely ignored even if you explicitly set it. A workaround is
 to edit your "generated/build/BuildConfig" file and change:

 HAIKU_HOST_USE_32BIT                            ?= "0" ;

 to:

 HAIKU_HOST_USE_32BIT                            ?= "1" ;

 This will fix the build.
