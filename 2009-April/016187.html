 The *.vmx file needed to start Haiku is incorrect IMO.

 I tested [http://haiku-files.org/vm/haiku-pre-alpha-r30085-vm.zip]

 The image file is "haiku-alpha.vmdk" whereas the vmx lists:

 ide0:0.fileName = "haiku.vmdk"

 Therefore it won't start automatically by executing the vmx.
