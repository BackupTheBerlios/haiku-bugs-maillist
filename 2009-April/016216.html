Comment(by oco):

 From my understanding, those messages come from consistency checks when
 loading termcap file.

 Here is what i have done to workaround the problem in offending entries :

  - put im entry before ie entry (fix "exit_insert_mode but no
 enter_insert_mode")
  - put empty ts entry (ts=:) when the existing one is commented (with ..)
 (fix "from_status_line but no to_status_line")
  - put empty Ip entry(Ip=:) when the existing one is commented (with ..)
 (fix "set_color_pair but no initialize_pair")

 Maybe there is a way to disable those warnings in the code to avoid this.
