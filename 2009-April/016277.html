Comment(by mmlr):

 The correct way to do this is to use the build platform built makebootable
 using the jam run target like this:

 {{{
 jam run ":<build>makebootable" /dev/disk/path/to/partition
 }}}

 This works under Haiku and should also work under BeOS.
