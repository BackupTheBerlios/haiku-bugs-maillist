Comment(by haiqu):

 I just did a clean build, and it still isn't right as at r30226. What's
 needed in the scripts is:

 {{{
 link from boot/system/lib/libbe.so -> boot/develop/lib/x86/libbe.so
 link from boot/system/lib/libroot.so -> boot/develop/lib/x86/libroot.so
 link from boot/system/lib/libstdc++.r4.so ->
 boot/develop/tools/gnupro/lib/libstdc++.r4.so
 }}}

 Then it's all good and the Haiku sources build with the native compiler,
 as long as ./configure is done correctly.
