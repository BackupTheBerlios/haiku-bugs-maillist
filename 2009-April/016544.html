 There are a number of GPL-licensed binaries at haiku-files.org that
 currently come without source code or offer of source code, we need to
 rectify that.[[BR]]
 Just putting the source code up for download is probably the path of least
 resistance, IMO. Note that merely linking to upstream sources, even for
 unmodified code, is probably not advisable (see FSF/Mepis spat http://www
 .linux-watch.com/news/NS9113053102.html).
