Comment(by idefix):

 Replying to [comment:4 haiqu]:
 > OK, that needs clarifying. If I go into DriveSetup and try to initialize
 that partition, the option is greyed out. I had no such problem with
 Partition 3 but, if you look at the first snapshot you can see that it's
 still narked in the partition table as type 06 (fat16, dos 3.31) [...]
 I think this is the cause of the problem:[[BR]]
 Apparently Installer only looks at the partition table and sees a fat16
 partition, which has a maximum size of 2 GB.[[BR]]
 Tracker, on the other hand looks at the information inside this partition
 and sees a fat32 formatted one, which can be larger than 2 GB. So it
 reports the correct partition size.

 I suspect that changing the type of this partition in the partition table
 to fat32 would solve your problems.
