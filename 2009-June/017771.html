 When packaging the Free Pascal Compiler today I was using a spare empty
 partition to set out all the files to be included. Some of them needed to
 be in the Home directory, because that's where the compiler looks for
 configuration information if it isn't in the current directory.

 When I zipped up the package I noticed the file was much larger than the
 last build, and on investiagtion discovered that Zip-O-Matic had included
 the hidden directory ~/Desktop/Trash and all its contents.

 I've listed this under Applications/Tracker even though it's an add-on
 because Zip-O-Matic doesn't seem to have its own entry.
