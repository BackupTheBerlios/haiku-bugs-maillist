 This is r31031.

 You need a horizontally very large Keymap window to display all keys
 untruncated. I'd propose to use abbreviations where possible:

 CAPS LOCK -> CAPS
 NUM LOCK -> NUM
 CONTROL -> CTRL
 OPTION -> OPT
 COMMAND -> COM
 PAGE ↑ -> PG ↑
 PAGE ↓ -> PG ↓

 It'd also be a good idea to somehow get rid of the list views to the left.
 But where to put it...? Below the keymap? Dunno...
