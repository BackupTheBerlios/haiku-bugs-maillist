Comment(by stippi):

 Since Francois fixed the BeOS NetSurf build in SVN, I've been able to
 build NetSurf on Haiku. The issue with the error message about ./beos/res
 happens when the NetSurf resources are not installed to
 '''/boot/home/config/settings/NetSurf'''. In the source, look into
 '''beos/res''', there are a bunch of links to resource files in other
 parts of the source tree. Just copy the originals with the same name as
 the link into the settings folder. Obviously, it also needs to be part of
 the package.
