Comment:

 I think that tickets about Haiku not working on specific hardware should
 not be in the R1/alpha milestone, because there are great chances of
 always finding some specific hardware that Haiku does not boot on. The
 hardware for which this ticket has been initially posted for actually
 works now. Other hardware does not. I believe that only issues which
 affect all hardware should be in R1/alpha. Like for example missing ATAPI
 support in the new ATA bus manager. But it's not clear right now if we
 really go with that driver, since work on it is currently on hold.
