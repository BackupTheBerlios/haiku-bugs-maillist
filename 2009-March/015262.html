 In Icon View (and Mini-Icon) mode, Tracker should auto arrange icons when
 the windows is resized. This way all icons could be viewed, fitting in the
 windows, instead making the user to scroll or to move them manually.

 In r29618 (as in BeOS R5), icons are always still. Doing a Clean Up
 (Alt+K) won't help in this case. Only doing Alt+Shift+K would do it, but
 this feature is hiddem (thanks [http://betips.net/1997/09/09/clean-up-
 your-icons/ Betips]!). In fact, if auto arrange is implemented, we could
 eliminate Clean Up and its variations.
