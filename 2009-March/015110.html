Comment(by bonefish):

 That's not a bug, but a feature. Not keeping the disk systems loaded would
 mean that for every partition operation you'd have to load and unload the
 respective disk system.

 Replacing disk systems is intended to work based on node monitoring. All
 partitions associated with the disk system in question would need to be
 invalidated and rescanned. Care must be taken regarding mounted and busy
 partitions.

 Whoever added node monitoring to the DDM apparently forgot to add that
 feature. :-)
