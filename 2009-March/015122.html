 Checking the options "Automatically expand files" and "Close window when
 done expanding" leaves the app not easily available to the inexperienced
 user to revert these settings or any options from the preferences panel.

 He has probably always just double-clicked an archive to start Expander.
 Now it will always expand automatically and quit when done.

 The solution is of course to start Expander without an archive. Easy: just
 start it from /boot/beos/apps/Expander or enter "Expander" in Terminal.
 Joe User maybe doesn't know that, however...

 So, back into Deskbar's "Applications" menu?
