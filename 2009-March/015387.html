Comment(by axeld):

 It's actually wanted that an application can override this key. Just think
 of emulators such as Qemu - if you press the Print Screen key there, you
 should actually expect it work in the emulator, not outside of it.

 Every key shortcut that is not handled within the application is
 impossible to override - while this might be desirable for things like
 changing a workspace (so that an app cannot lock you into a specific
 workspace), most key shortcuts should be handled within the application
 when possible.
