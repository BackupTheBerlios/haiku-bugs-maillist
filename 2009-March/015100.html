 Hi!

 The gcc that comes with Haiku does not support Objective C. This is not a
 big problem atm, as there aren't many (any?) Objective C apps for Haiku
 yet, but it won't hurt to enable it. All you need to do is change the
 configure line of gcc and add objc to --enable-languages there (currently
 you have --enable-languages=c,c++, you could change it to either --enable-
 languages=c,c++,objc,obj-c++ or --enable-languages=c,c++,objc. Including
 obj-c++ would be better so that bindings for Haiku's C++ APIs could be
 written). Reason is that I like Haiku and I'd like to port my Objective C
 Framework Library to it so that applications using it run out of the box
 on Haiku.
