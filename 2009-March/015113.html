Comment(by BeOSR):

 The indicated size would be limited to the *active window only*, and not
 traverse (sub)directories.
 If you open a window, the size of any folder indicated as "-", just as it
 is for symlinks. These would be counted at that value, so zero. Only
 actual files or objects would be totaled.

 This is not really relevant for system folders, but is very handy for user
 folders.

 For an overview of used disk space per folder, DiskUsage works very well.
