 When comparing keyboard navigation between Single and Multiple window
 navigation,  opt (winkey)+alt+down functions as expected in multiple
 window navigation, yet is functionless in single window mode.

 r29476

 Single window:
 alt+up: navigate to parent
 alt+down: navigate to selected folder
 opt+alt+up: new window of parent
 opt+alt+down: nada ,

 Multi-window:
 alt+up: new window of parent
 alt+down: new window of selected folder
 opt+alt+up: navigate to parent
 opt+alt+down: navigate to selected folder
