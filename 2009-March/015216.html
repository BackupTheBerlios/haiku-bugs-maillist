 This camera has an internal storage and a SD card can be plugged in. There
 is no way that the cameras internal storage or SD card can be mounted to
 copy photos by Tracker or no application that can detect the camera and
 import photos.
 I tried Exposure (available from Haikuware) but there is no plugin for
 this application for the Kodak C1013. Also the installer for this
 application does not work in Haiku. It installs in Beos5.05Bone alright.
 The only application that works well with this camera in Linux is: f-spot
 that comes with GNOME. this application lists it as a PTP type camera when
 it is plugged into a USB socket and can import photos when this is
 selected.
 So it might be possible to use the source for f-spot to make a plugin for
 Exposure.
 I looked at the developers web site for Exposure and he is no longer
 developing it. I emailed him but he hasn't replied. The developers web
 address is:
 http://www.gwsoft.co.uk/exposure/
