 Using VMware Player for Linux, the screen (vmware window content) is no
 longer refreshed once KDL has been entered. I see the initial few lines
 of the Welcome to kernel debugging land, but every command I enter appears
 to be ignored, as the screen doesn't change.
 However, when quitting (suspending?) the vmware session, and reopening
 it, the modified KDL screen, including the commands that I typed and it's
 output are visible. Still the screen doesn't change, unless I quit vmware
 again.

 Observed with a r19974 build form haikuhost.com and a self build r19981.
