Comment (by axeld):

 The error you see is 0x80007029 (B_BUFFER_OVERFLOW) which means as much as
 that the array you passed in is not large enough to hold the page
 descriptions for the memory block you asked for.
 Since the code in if_em.h only passes in a single array entry: are you
 sure you only ask for memory on page boundaries? Because if not, the code
 is wrong, and the error returned is a good sign for this.
