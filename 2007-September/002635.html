 Tested with build r22191.

 Open any folder.
 When resizing the Tracker window horizontally, the drag icon in the menu
 bar at the top right disappears.
 I made a small hey script where I resize the window horizontally pixel by
 pixel. Move one pixel and the icon disappears. After another 18 pixels,
 the icon reappears. One pixel farther, the icon disappears again, only to
 again reappear after another 18 pixels.
 Then I modified the same script to shrink the window again, one pixel at a
 time. Now the icon does not reappear! MIA.

 Maybe that isn't even Tracker's bug, but belongs to the interface kit that
 has seen some changes for menus recently.
