 There is a serial debug feature in EHCI chipsets, that works using a small
 set of PCI registers, bypassing the USB stack, and device hierarchy. How
 ever it requires a special adapter (don't know where you get them).

 http://linuxbios.org/index.php/EHCI_Debug_Port

 Alternatively the parallel port could be used (again if there is one).
 Quite why laptops often drop serial, yet keep parallel I don't know.
