 Another certain way to cause this bug is to copy a wav file from an audio
 CD on to the other beos partition. Exit from the Tracker window. Unmount
 the beos partition. KDL is the result.
 So far all the repeatable ways I have found involve writing files to the
 other beos partition. The files have to be a reasonable size. Not just a
 few kb's as in a text file (exception to this was downloading a few 100
 kb's in beshare) but a few mb's.
