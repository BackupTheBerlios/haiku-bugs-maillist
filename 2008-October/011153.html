 get told that a file of name "b" already exists and two buttons, "Cancel"
 and "Replace other file".

 "b" to "a".

 However I can replace a folder with another folder and a file with another
 file.

 Expected behaviour: do the replace; however, replacing a folder with a
 file can be harmful: losing all the files in a folder. So perhaps the
 dialog box should contain a warning if I'm trying to replace a non-empty
 directory (regardless whether with a file or directory).
