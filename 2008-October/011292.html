 This is r28287.

 First off, ~/config/settings/Tracker/Go that contains links to your
 favorite folders, should be renamed to .../Favorites. It's more consistent
 with the Open/Save panel where you can set these.

 Furthermore, I'm for setting favorites like Tracker's "New..." templates,
 i.e. instead of a configuration panel simply have a "Edit Favorites..."
 that opens the favorites folder.

 The same goes for the panel "Configuring the Deskbar (Leaf) Menu" of the
 Deskbar. Do away with the configuring the be folder part. This can as well
 be dealt with opening the ~/config/be folder.

 Even better, since drag&drop of files works with the Deskbar to add
 entries, offer a "Remove from Deskbar" when right-clicking on an entry in
 the Deskbar. Then we won't have to open the be folder at all for
 configuring.

 This leaves the number of Recent Apps/Folders/Documents which can be dealt
 with a long overdue Deskbar preferences panel. This setting doesn't have
 to be configurable from Open/Save panels.
