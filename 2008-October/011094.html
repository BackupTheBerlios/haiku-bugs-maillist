Comment(by andreasf):

 My git patch was a quick addition to get the revision number displayed. A
 better solution might be to split up the command, e.g. (pseudocode)
 {{{
 revision=`(svn info || git svn info || other-svn-info-compatible-VCS ||
 echo "Revision: 0") | grep Revision | awk ...`
 if (revision == "0")
     revision=`git log ... | echo 0` # try git without git-svn
 if (revision == "0")
     revision=... # try Mercurial/Bazaar/your-favorite-VCS
 }}}
 That would avoid that use of `awk` to put it into the {{{Revision:
 nnnnn}}} format to later extract it again.
