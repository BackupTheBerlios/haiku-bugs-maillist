 I recently reported here:

 [http://www.haikuware.com/20081013249/wesnoth-on-haiku]

 That Wesnoth was running on Haiku.

 The original port was done by B. Korz and crew on Zeta.

 I tested the game on BeOS, and it behaves as expected, except no
 networking (for multiplayer).

 I then tested it on Haiku.

 There's three issues:

 1) The binary can't be executed unless it's from a terminal (yes, the
 permissions are correct). On BeOS, it's no problem.

 2) Having your monitor set to the default resolution for Wesnoth
 (1024x768), the game starts in full screen under Haiku, but then kicks
 back to windowed mode (under BeOS it stays in full). When you uncheck
 toggle fullscreen in the video preferences, it goes back to fullscreen,
 but it's pitch black.

 3) Networking doesn't work, yet the errors are different. Under BeOS, the
 error will be 'Could not connect to host' no matter what you try, whereas
 under Haiku the error is 'Illegal character in compression input' - see
 attached.

 The game, v1.2.1, can be found here:

 [http://www.haikuware.com/view-details/games/strategy/wesnoth12]

 Please read the specific instructions under the file's description to
 start the game. Currently you have to put a comment of false under full
 screen in the preferences file it creates in order to play it properly.

 Would be nice to see this game working well under Haiku.
