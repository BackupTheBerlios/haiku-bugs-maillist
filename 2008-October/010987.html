 This is revision r28069.

 The folder {{{/boot/home/config/settings/Tracker/QueryResultTemplate/}}}
 offers the possibility to assign a certain attribute layout for the result
 window of a query for a specific filetype.
 To do so, is a bit awkward, however: You have to create a folder, rename
 it to the filetype’s mime string, replace slashes with underscores and
 copy a file of that type to it to arrange the attribute columns…

 How about, if you query for a *specific* filetype (a query for a filename
 is too generic for saving a layout making much sense), the result window
 offered a new menu item “Attributes | Save Layout” that would do the above
 for you?
