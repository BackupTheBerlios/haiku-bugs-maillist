 Now it's possible to do:[[BR]][[BR]]
 jam haiku-image[[BR]]
 sudo dd if=generated/haiku.image of=/dev/disk0s3 bs=128m count=1 (or
 whatever spare partition you have)[[BR]]
 sudo jam run ":<build>makebootable" /dev/disk0s3[[BR]]

 reboot and boot from it using rEFIt on intel macs (it still crash on boot,
 ehci cleanup thread and others (i think macbook webcam causes it), but
 that is another issue :)
