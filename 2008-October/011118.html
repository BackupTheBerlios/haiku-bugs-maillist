 Talk Box:

 [http://www.haikuware.com/view-details/multimedia/audio/utilities/talkbox]

 Installs correctly, yet the binary can't be executed normally (I had this
 issue with Wesnoth as well), but can be executed from the terminal. Only
 problem is, you have to drag and drop text files on the Talk Box icon, so
 that doesn't work in Haiku.

 Anyways, I tested it on BeOS, and it works fine. You can test it on Haiku
 by executing the 'Change Voice' binary. It will speak out some text for
 the different voices, '''only on Haiku these are scratching noises''',
 whereas on BeOS it actually speaks.

 A good test for the media kit. Would be nice if this program were opened,
 and could somehow be adapted as an optional package for Haiku
 'accessibility features' i.e visually impared.
