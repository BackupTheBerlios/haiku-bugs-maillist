Comment(by bonefish):

 Replying to [comment:18 anevilyak]:
 > How does one go about setting kernel watchpoints? Will check the other
 items in your list after work.

 First of all enable them (KERNEL_BREAKPOINTS in kernel_debug_config.h).
 Then either use the kernel debugger command "watchpoint" -- {{{watchpoint
 <address> 4}}} sets a 4-byte write watchpoint -- or set them
 programatically, i.e. include <arch/user_debugger.h> and call
 {{{arch_set_kernel_watchpoint(address, B_DATA_WRITE_WATCHPOINT, 4);}}}.
