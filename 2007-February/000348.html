Comment:

 How will we handle the case when someone adds more workspaces, goes to a
 new workspace, changes the resolution, and then clicks Revert? IMHO, it
 would be very bad to switch the user to some other workspace (esp. without
 him noticing). The same applies to Defaults.

 What speaks against having a fixed number of workspaces (e.g., 12, one for
 each F-key)?

 If that is not an option we could move all buttons into the screen prefs
 box, so they are visually separated from the workspace settings. This
 would be merely a "hack", though...
