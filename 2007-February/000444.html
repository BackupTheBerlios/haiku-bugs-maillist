 File => Save as Draft is always enable,
 even when there is no changes to the text,
 or the Subject, or the <To> field.

 Open BeMail and create a new mail.
 Fill <To>
 Fill <Subject> with bar (or anything you like).
 Write a line to the body text.
 Then File => Save as Draft
 Close BeMail.

 Open bar with no changes at all
 File => Save as Draft is always enable.
