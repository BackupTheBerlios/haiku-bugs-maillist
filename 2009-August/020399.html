 This patch adds support for the for BSD-style xattrs interface (which is
 almost identical to Linux's attr/xattr.h, except in regard of symlink
 traversing options)

 It allows building Haiku on Mac OS X with --use-xattr from a ZFS volume
 (HFS+ does not work though).
