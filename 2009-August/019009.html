 According to BeOS texts, 'query' should be able to parse dates and times
 in several different formats. This does not appear to be the case with
 Haiku's 'query' tool.

 Examples:

 1. last_modified > 08/01/09

 2. last_modified > August 1, 2009

 3. last_modified > 08012009

 None of the above queries work as intended, and return unpredictable
 results.

 NOTE: This page details several query formats:
 http://www.birdhouse.org/beos/bible/exc_query.html
