 If a kernel driver for a graphics chip is present and publishes a device,
 but its corresponding accelerant is missing, the app_server fails to
 gracefully fall back to a different device. Noticed this by accident while
 disabling the radeon (removed the accelerant instead of the kernel driver
 like I usually do). End result, "Could not initialize graphics output,
 exiting" on serial, and the Desktop never appeared.
