 The AboutSystem dialog displays AMD Athlon 64 on my  Sempron 64

 Also I had previously thought this was a 32bit cpu but cpuinfo says
 otherwise since it has the lm flag which is discribed in the cpufeature.h
 header of the Linux kernel (learned that much from Google anyway)
 http://www.gelato.unsw.edu.au/lxr/source/include/asm-i386/cpufeature.h

 This has been this way for a long time and still there as of r32416
