 If a newcomer boots the CD to desktop instead of to installer - perhaps
 looking for a quick preview of Haiku - it's not easy to find Installer
 unless you know what you're looking for and where to look. (Deskbar/Find
 did not work on the CD when I tried it tonight.) A link to Installer on
 the desktop and/or in Deskbar would help many newcomers avoid a second,
 painful CD boot.

 This is what I suggest for Deskbar:
 {{{
 Applications
 Demos
 Desktop Applets
 Preferences
 Installer // sorted after folders, since it's an application
 }}}


 We want people to try Haiku off harddisk and get a better impression of it
 than what the CD offers. Placing it this prominently will get more people
 to install to harddisk sooner.

 A post-install script could remove it from Deskbar.
