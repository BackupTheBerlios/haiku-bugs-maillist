Comment(by stippi):

 I would change the interface for Query windows. I would combine the Find
 panel and the Query result window. Make the window title "Unsaved Query"
 for as long as it's not saved, "Temporary Query". Make it possible to
 collapse the Find area where you specify the parameters. That way you show
 information about the query in the same window, while not resorting to the
 window title. And it would be much less awkward to refine the query. I
 always didn't like very much how that worked, way too clumsy.
