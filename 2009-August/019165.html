Comment(by pulkomandy):

 This is one of the reason I wanted to centralize some strings in a common
 set and not let the dev choose them freely. In this system there would be
 a #define B_CANCEL_BUTTON "Cancel" (or something else) set by each locale.
 This way it is easy to identify which button is "ok", which one is
 "cancel" and so on. Allowing you to detect the right button every time,
 whatever the locale is, and without adding too much code.
