Comment(by mmlr):

 Do you boot that external drive using an older bootloader? If so it is
 likely that you have an incompatible version of the bootloader. There was
 a recent change to one of the structs passed from the bootloader to the
 kernel. If you're using an old bootloader and try to boot a new kernel you
 will see such reboots.
