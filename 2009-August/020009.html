 I was doing a new Haiku installation and this involved copying hundreds of
 thousands of files from the old partition to the new one.   My first
 attempt to do the copy resulted in a crash but I have a USB keyboard so I
 could not get a stack crawl. I reformatted the partition and tried again
 taking care to intentionally enter KDL once so the keyboard would be
 enabled if I ended up in KDL again.
