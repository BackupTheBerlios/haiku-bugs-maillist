 The context is the RTF DataTranslator, line 622 of convert.cpp in src/add-
 ons/translators/rtf/

 Instead of operating on BPositionIO, the translator casts it into a BNode.
 The downside of this is that it is impossible to use any subclass of
 BPositionIO except BFile with the translator. Using any subclass of
 BPositionIO except for BFile returns I/O error (-2147483647).
