 FilePanel should also provide the popup menu that appears when "<n> items"
 in the bottom left of a Tracker window is clicked. This makes the drop-
 down list that sits above the file list view obsolete. The latter is less
 versatile, afterall.

 I would personally also find it handy to have the Tracker navigator at the
 top of FilePanel, so that one could quickly navigate the file system with
 the help of the buttons or enter a path by keyboard.

 Adding Tracker's "Window" menu with some of its items might also be
 helpful. Especially when a thumbnails view mode might get added in the
 future.
