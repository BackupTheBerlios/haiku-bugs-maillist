#1439: Audio CD not refreshing if CD is changed
--------------------------+-------------------------------------------------
 Reporter:  scottmc       |       Owner:  axeld         
     Type:  bug           |      Status:  new           
 Priority:  normal        |   Milestone:  R1            
Component:  File Systems  |     Version:  R1 development
 Platform:  All           |  
--------------------------+-------------------------------------------------
 Upon booting Haiku it correctly detects an audio CD.  I open the CD and
 drag some of the wav files to the desktop.  Then eject the CD and insert a
 different audio CD.  The audio CD still shows the wav files from the
 previous CD.  If you drag any of the wav files from that second CD to the
 desktop they "seem" to copy from the same part of the CD that the file
 would have been from on that first CD.  Reason I say this is if you then
 try to play that wav file from the second CD it starts playing one song
 and then goes into part of another.  Seems like it's just not updating the
 CD info when a disc is changed.
