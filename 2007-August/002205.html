 The outer BView::EndPicture() returns NULL!

 Sample code:
 BView *view = ...
 view->BeginPicture(new BPicture());
 ...
   view->BeginPicture(new BPicture());
   BPicture *nestedPicture = view->EndPicture();
 ...
 BPicture *picture = view->EndPicture();
 // picture should be the BPicture object created at the beginning

 See FlattenPictureTest application
 src/tests/kits/interface/flatten_picture
