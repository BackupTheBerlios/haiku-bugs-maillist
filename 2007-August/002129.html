#1365: broken isinf in math.h
-----------------------+----------------------------------------------------
 Reporter:  kaliber    |       Owner:  axeld         
     Type:  bug        |      Status:  new           
 Priority:  normal     |   Milestone:  R1            
Component:  - General  |     Version:  R1 development
 Platform:  All        |  
-----------------------+----------------------------------------------------
 There is:


 {{{
 # define isinf(x) \
     (sizeof(value) == sizeof(float) ? __isinff(value)           \
         : sizeof(value) == sizeof(double) ? __isinf(value)      \
         : __isinfl(value))

 }}}

 but should be:

 {{{
 # define isinf(value) \
     (sizeof(value) == sizeof(float) ? __isinff(value)           \
         : sizeof(value) == sizeof(double) ? __isinf(value)      \
         : __isinfl(value))
 }}}
