 FreeBSD cannot build Haiku due to a problem with Yacc/Bison/Bison2/etc.

 This patch makes the build system use 'bison2' instead of bison/yacc.

 With this patch FreeBSD (v6.2/STABLE) is able to compile a haiku image
 without any problems.

 PS: Should a list of the ports need to be installed on FreeBSD be added to
 the cross compiler readme file?
