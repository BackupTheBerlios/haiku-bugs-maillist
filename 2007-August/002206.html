 At the moment, each email to the bug list is sent by “Haiku <trac at haiku-
 os.org>”, which makes sorting through less than easy.

 It would be nice if Trac could send the message as “%username%
 <trac+%username%@haiku-os.org>”.
 This would not affect the current behaviour, but improve readability.

 How it is now (in gmail):
 Bug #1368 - Haiku, Haiku, Haiku, Haiku, Haiku, Haiku, Haiku
 How it could be:
 Bug #1368 - rdaneel, axel, korli, bonefish, korli, korli, bonefish

 Not really important now, but will be in the future as the list becomes
 busier.
