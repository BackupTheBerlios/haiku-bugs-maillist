 I'm currently building a screensaver which uses Haiku icons, I hope we
 didn't have the same idea ;-p[[BR]]
 If you have integrated your code to the Haiku build system, using
 IconsUtils.h is pretty straightforward. Here is what I used:

 {{{
 SubDir HAIKU_TOP src add-ons screen_savers fallingicons ;

 UseLibraryHeaders icon ;

 ScreenSaver FallingIcons :
         ConfigView.cpp FallingIconsSaver.cpp FallingIcon.cpp :
         be libscreensaver.so libicon.a $(TARGET_LIBSUPC++)
 ;

 }}}
