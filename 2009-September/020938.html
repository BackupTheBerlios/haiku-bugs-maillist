Comment(by scottmc):

 I've seen many cases where StyledEdit would allow editing of files that Pe
 said were read-only.  This happens quite a bit when porting apps and
 needing to edit configure scripts.  In Pe there is a graphical button
 showing a pencil, if it thinks the file is read-only that button will show
 a red X on it.  Click that button and now you should be able to edit the
 file.
