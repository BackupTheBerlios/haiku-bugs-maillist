 I dual-boot BeOS and Haiku now, and I can't seem to figure out how to get
 the mouse to feel the same in both places.

 In BeOS, I have the mouse speed set at the second tick from "slow", and
 acceleration just beyond one tick from "fast".  This allows single-pixel
 precision if I move the mouse slowly, as well as hitting any corner of the
 screen by moving at a medium to medium fast pace.

 In Haiku I put the speed to the fastest one that still allows single-pixel
 movements (I think it's somewhere in the middle) and max out the
 acceleration, and still the acceleration is too slow, so in order to not
 be snapping my wrist a lot I have to put the speed faster than single-
 pixel, which is annoying when you're doing something like selecting text
 with the mouse on a web page that uses a font where the letter I is one
 pixel wide.

 Is there some way to make it behave exactly like R5?
