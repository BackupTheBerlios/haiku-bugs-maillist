 ''Click on the fifth item with Ctrl held''

 In Haiku, the default shortcut key is ALT, not CTRL. This can be changed
 in the Keymap Preferences app. It works on a standard install by using the
 alt key instead of the ctrl key.



 *Click on the first item while holding down Shift[[BR]]

 *Click on the third item with shift held[[BR]]

 *Click on the fifth item with '''ALT''' held[[BR]]
