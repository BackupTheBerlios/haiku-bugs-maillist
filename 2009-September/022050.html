 It seems as though some packages like BeZilla and subversion have been
 patched to use B_USER_SETTINGS_DIRECTORY to store their configuration.
 Other packages like cvs, less, bash, vim, and others are still using $HOME
 to store things.  It would be nice to clean these up and have it
 consistent.
