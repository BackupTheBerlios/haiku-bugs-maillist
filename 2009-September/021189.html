 For web pages that get split into subpages, how about a link to show all
 pages as one long page?

 I often find subpages to be more of a speed bump than an aid to me. On a
 lot of websites - especially web forums - I find myself pre-loading as
 many pages as I can, as tabs. I like how it allows me to not have to wait
 for the next subpage to load. But I still find it annoying, and I'm sure
 it puts more strain on the web server.
