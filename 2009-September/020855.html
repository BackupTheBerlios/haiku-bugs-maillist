 Haiku does not support the UserShutdownScript and UserShutdownFinishScript
 mentioned in the Haiku User Guide:
 http://haiku.it.su.se:8180/source/raw/docs/userguide/en/bash-
 scripting.html

 Haiku does not have the system ShutdownScript and ShutdownFinishScript:
 http://haiku.it.su.se:8180/source/xref/data/system/boot/

 I've tried providing these 4 scripts but they don't get run by anything,
 from what I can tell.

 Maybe it's better to make Registrar launch the user scripts itself,
 omitting the system shutdown scripts.
