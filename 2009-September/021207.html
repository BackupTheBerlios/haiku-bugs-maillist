 The subject says it all.

 To reproduce open the Font preferences, change the Plain Font to something
 that is not Deja Vu Sans Book, then either open Terminal or open a new
 Terminal window if it is already running. It will be using that font and
 not the fixed width font, which causes strange display and prompt issues
 (which is really another issue with Terminal and non-fixed width fonts.)

 The Terminal preferences still shows the font as Deja Vu Sans Mono.

 This can be fixed for alpha2 and I will probably investigate it.
