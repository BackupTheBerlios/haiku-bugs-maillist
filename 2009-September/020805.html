Comment(by luroh):

 Well, a problem with that is that the vmx file available at haiku-
 files.org requires additional hackery to work with our alpha vmdk images.
 Specifically, it requires the user to edit the ide0:0.fileName parameter
 to match that of the vmdk file, else a "File not found: haiku.vmdk" error
 popup will be displayed.

 A possible workaround would be to change the name of the image from haiku-
 alpha.vmdk to haiku.vmdk.
