 Use the ICMP protocol to implement path MTU discovery as described in
 [http://tools.ietf.org/html/rfc1191 RFC 1191].

 Also interesting probably are [http://tools.ietf.org/html/rfc879 RFC 879],
 and even more [http://tools.ietf.org/html/rfc2923 RFC 2923] which
 discusses problems of TCP with path MTU discovery.
