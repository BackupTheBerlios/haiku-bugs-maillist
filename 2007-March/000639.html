 Platform:  All                    |

 Running programs on file systems not using the file cache fails. The
 runtime loader uses sys_vm_map_file() to map the executable file, which
 apparently doesn't work for FSs not using the file cache. _vm_map_file()
 (vm.cpp) has a TODO stating something to that effect.
