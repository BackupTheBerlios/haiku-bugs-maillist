Comment:

 Looks fine. The very first device listed is a Microsoft HID device. It has
 two interfaces of which the first one is the keyboard supporting the boot
 protocol. The second interface could very well be the mouse, or a combined
 mouse/keyboard interface. Since the second one doesn't work with the boot
 protocol (as it can't when providing more flexible input), it isn't
 recognized by the Haiku usb_hid, as this one currently still lacks generic
 HID support. Support for that is tracked in #2253, so closing this as a
 duplicate.
