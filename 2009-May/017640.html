Comment(by rudolfc):

 Hi,

 To loose the problem, you could place the file nvidia.settings in your
 home folder, and modify it in the line that reads:
 force_ws true
 to read:
 force_ws false

 If you now reboot, you should be able to use all normal 4:3 resolutions.
 Can you confirm this please?

 This problem will be gone once DDC is in use. It will probably (at least
 partially) solve these other bugs too.
