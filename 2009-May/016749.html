 kicks in. This modeswitch needs to set the your native resolution as you
 pointed out.

 compared to the BeOS version: it overwrites your VESA settings file after
 exiting this app. If you would want to test with a lower resolution on the
 panel after bootup: you probably need to set (for instance) 1024x768 with
 Screen: exit it, then modify the vesa file to read 1400x1050 Again, then
 do the reboot.

 Would that work? (or, probably in other words: if you switch down from
