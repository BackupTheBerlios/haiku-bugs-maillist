Comment(by kaliber):

 The problem is that key codes (in src/apps/terminal/VTKeyTbl.c) are
 adjusted for beterm. Unfortunately the default TERM variable is set to
 xterm, so key codes should be as follow:


 {{{
 char *function_key_char_table [] =
 {
 "\033OP",
 "\033OQ",
 "\033OR",
 "\033OS",
 "\033[15~",
 "\033[17~",
 "\033[18~",
 "\033[19~",
 "\033[20~",
 "\033[21~",
 "\033[22~",
 "\033[24~",
 };

 }}}
