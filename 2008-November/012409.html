 INADDR_ANY should be a compile time constant.

 haiku's netinet/in.h defines it as __IPADDR(0x00000000), where __IPADDR is
 ((uint32_t)htonl((uint32_t)(x))).

 ie., INADDR_ANY -> htonl(0x00000000), which is silly, and breaks some
 software (e.g. silc-client).

 FreeBSD just defines it as a straight constant, 0x00000000.
