 {{{
 BuildHaikuImage1 /dev/sda3

 I build from xubuntu-x64, sda3 is a 60GB partition and the added line to
 UserBuildConfig that fails the build below:
 {{{
     CopyDirectoryToHaikuImage home : /home/frho/projects/mozilla : : -L ;
 }}}
 Removing it causes the build to succeed and I have another similar line
 for a smaller dir that doesn't cause this.

 The mozilla dir was created by:
 {{{
 export CVSROOT=:pserver:anonymous at cvs-mirror.mozilla.org:/cvsroot
 export MOZ_CO_PROJECT=browser
 cvs co -r MOZILLA_1_8_BRANCH mozilla/client.mk
 cd mozilla
 make -f client.mk checkout
 }}}
