Comment(by bonefish):

 The first and the third problem are fixed in r31523 respectively r31524.
 The second one is a bit weird, since jam seems to know that builtins.o
 depends on builtins.c. "jam -dd" says so at least:
 {{{
 Depends "<src!bin!bash!builtins>builtins.o" :
 "<src!bin!bash!builtins>builtins.c" ;
 }}}
 So it shouldn't try to make builtins.o before builtins.c has been made.

 Anyway, I've built bash a few times with "-j8" and didn't get anymore
 errors. Please give the new revision a whirl.
