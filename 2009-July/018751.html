 In the Bebook, this function is described:
 status_t SetMode(display_mode* mode, bool makeDefault = false);
 It does not work in Haiku: the call is never made to the accelerant.

 What does work in Haiku is this (which is used in Haiku's screen preflet):

 SetMode(workspace, &mode, false);

 This problem exists for a long period of time now, upto/including R31786.
 Newer ones are not tested.
