 This is r31843. When trying to create a symlink on a filesystem that does
 not support them (i.e. the fs_create_symlink hook is NULL), EROFS  is
 returned. It should probably return B_UNSUPPORTED instead.
 {{{
 /fatfs> ln -s somefile somelink
 ln: creating symbolic link `somelink': Read-only file system
 }}}
